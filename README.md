# Card de Matrícula – Do Figma ao Código

## 📌 Sobre o projeto

Este projeto foi desenvolvido como parte da atividade prática **“Do Figma ao Código – Card de Conversão”**.
O objetivo foi criar um **card de matrícula para cursos online**, passando pelas etapas de **Wireframe, Protótipo de Alta Fidelidade e Desenvolvimento Front-end**.

O projeto aplica conceitos de **UX/UI Design**, **hierarquia visual**, **responsividade** e **boas práticas de desenvolvimento web** utilizando **HTML5 e CSS3**.

---

## 🎯 Objetivo

Criar um card de matrícula otimizado para conversão, que apresente de forma clara as informações de um curso e incentive o usuário a realizar a inscrição.

---

## 🧩 Etapas do Projeto

### 1️⃣ Wireframe

Estrutura inicial do card desenvolvida no Figma em baixa fidelidade.
Nessa etapa foram definidos:

* Posicionamento da imagem
* Tag de categoria
* Título do curso
* Descrição
* Preço
* Botão de ação (Call to Action)

O foco foi estabelecer a **hierarquia das informações** sem utilizar cores ou elementos visuais detalhados.

---

### 2️⃣ Protótipo de Alta Fidelidade

O wireframe foi transformado em um protótipo visual completo no Figma.

Foram aplicados:

* Paleta de cores
* Tipografia
* Contraste visual
* Princípios de Gestalt
* Estados de interação (hover no botão)

Essa etapa teve como objetivo criar uma interface **mais atrativa e intuitiva para o usuário**.

---

### 3️⃣ Desenvolvimento Front-end

O protótipo foi implementado utilizando **HTML5 e CSS3**.

Principais recursos utilizados:

* Estrutura semântica em HTML
* Flexbox para organização do layout
* Border-radius para suavização de elementos
* Box-shadow para profundidade visual
* Transições no botão para simular interação
* Responsividade com meta viewport

---

## 🛠 Tecnologias utilizadas

* HTML5
* CSS3
* Figma
* GitHub

---

## 📱 Responsividade

O layout foi desenvolvido pensando em **boa adaptação para diferentes tamanhos de tela**, mantendo a legibilidade e a organização dos elementos.
